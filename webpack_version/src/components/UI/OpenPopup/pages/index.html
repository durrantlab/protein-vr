Program Description

Protein Virtual Reality (Protein-VR) is an innovative tool for teaching micro and structural biology. Traditionally, instructors have used 2D schema that obscure the 3D nature of microscopic systems. Protein-VR leverages new and emerging technologies such as 3D web graphics and virtual-reality headsets to display these systems in full 3D.

Troubleshooting

If you wish to view Protein-VR scenes through a VR headset, your browser must support WebVR. Otherwise, you can still view the scenes using regular (2D) in-browser visualization. Voice commands are only supported in Google Chrome. We generally recommend Chrome over other browsers, running on Windows desktops or Android phones. If you're having trouble using Protein-VR, please click on your browser below for further instructions.
